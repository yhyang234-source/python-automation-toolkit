# 📚 교보문고 장바구니 자동화 봇 사용 설명서

---

## 📁 폴더 구성

실행 전 아래 파일이 같은 폴더에 있어야 합니다.

```
kyobo_bot/
├── kyobo_bot.py        ← 자동화 프로그램 (건드리지 마세요)
├── book_list.xlsx      ← 신청 도서 목록 (매번 교체)
└── README.md           ← 이 파일
```

---

## 🖥️ 처음 사용하는 PC 설정 (최초 1회만)

### 1단계 — Python 설치

1. 아래 주소에서 Python 다운로드
   - https://www.python.org/downloads/
2. 다운로드된 설치 파일 실행
3. 설치 첫 화면에서 **"Add Python to PATH"** 반드시 체크

   ```
   ☑ Add Python to PATH   ← 이거 꼭 체크!
   ```

4. **[Install Now]** 클릭 후 설치 완료까지 대기

---

### 2단계 — 필수 패키지 설치

1. 키보드에서 `Win + R` 누르기
2. 열린 창에 `cmd` 입력 후 확인
3. 검은 창(명령 프롬프트)에 아래 명령어 입력 후 Enter

```
pip install pandas selenium tqdm openpyxl
```

4. 설치가 완료될 때까지 대기 (인터넷 속도에 따라 1~3분 소요)
5. 오류 없이 완료되면 창 닫기

---

## 🚀 매번 사용할 때

### 3단계 — 도서 목록 준비

`book_list.xlsx` 파일에 아래 컬럼(머릿글)이 있는지 확인하세요.

| 컬럼명 | 내용 |
|---|---|
| `신청 책 제목(本タイトル)` | 도서 제목 |
| `작가(作家)` | 저자명 |
| `출판사(出版社)` | 출판사명 |

> 제목이 비어있는 행은 자동으로 건너뜁니다.

---

### 4단계 — 크롬 디버그 모드 실행

> ⚠️ 봇은 현재 열려있는 크롬에 접속하는 방식입니다.
> 반드시 아래 순서대로 크롬을 열어주세요.

1. 현재 열려있는 **크롬을 모두 닫기**
2. `Win + R` 누르고 아래 명령어를 **복사해서 붙여넣기** 후 확인

```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\chrome-debug-profile"
```

3. 열린 크롬에서 **교보문고 로그인** 완료
   - https://www.kyobobook.co.kr

---

### 5단계 — 봇 실행

1. `Win + R` 누르고 `cmd` 입력 후 확인
2. 아래 명령어로 kyobo_bot 폴더로 이동 (폴더 경로는 본인 환경에 맞게 수정)

```
cd C:\kyobo_bot
```

3. 아래 명령어 입력 후 Enter

```
python kyobo_bot.py
```

4. 진행 상황이 화면에 표시됩니다

```
진행:  45%|██████████░░░░  | 32/71 [03:12<03:54]
[14:32:06] INFO —   ✔ 성공 — 프로젝트 헤일메리
```

5. **"모든 자동화 작업이 완료되었습니다"** 메시지 확인 후 종료

---

## 📊 결과 확인

작업 완료 후 같은 폴더에 아래 파일이 생성됩니다.

| 파일명 | 내용 |
|---|---|
| `result_report_final.xlsx` | 전체 처리 결과 |
| `failed_report.xlsx` | 실패 항목만 별도 정리 (수동 처리용) |
| `automation.log` | 상세 실행 로그 |

### CHECK 컬럼 상태값

| 값 | 의미 |
|---|---|
| `완료(성공)` | 장바구니 담기 성공 ✔ |
| `실패(검증불일치)` | 제목/저자가 달라서 담지 않음 |
| `실패(검색결과없음)` | 검색 결과 없음 |

## ❓ 자주 묻는 질문

**Q. `python`을 입력했더니 "인식할 수 없는 명령어"라고 나와요.**  
A. Python 설치 시 "Add Python to PATH" 체크를 빠뜨린 경우입니다. Python을 다시 설치해 주세요.

**Q. 크롬이 열리지 않아요.**  
A. 크롬 설치 경로가 다를 수 있습니다. 아래 경로로 변경해서 시도해 보세요.
```
"C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\chrome-debug-profile"
```

**Q. 실행 중 크롬을 건드렸더니 오류가 났어요.**  
A. 봇 실행 중에는 크롬을 직접 조작하지 마세요. `python kyobo_bot.py`로 다시 실행하면 됩니다.

**Q. 특정 책만 계속 실패해요.**  
A. `failed_report.xlsx`의 `상세사유` 컬럼을 확인하세요. 해당 항목은 교보문고에서 직접 수동으로 담아주세요.
