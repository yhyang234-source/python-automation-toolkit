==================================================
        표창장 자동 생성 시스템 사용 설명서 (v2.0)
==================================================

이 프로그램은 엑셀(Excel)에 정리된 대상자 명단을 읽어와, 
한글(HWP) 표창장 템플릿에 자동으로 데이터를 입력하고 
개별 파일로 생성해 주는 자동화 프로그램입니다.

아래 순서대로 천천히 따라해 주세요.

--------------------------------------------------
1. 파이썬(Python) 설치하기 (최초 1회만 진행)
--------------------------------------------------
이 프로그램이 돌아가기 위한 '엔진'을 설치합니다.

① 인터넷 창을 열고 아래 주소로 접속합니다.
   https://www.python.org/downloads/
② 노란색 [Download Python 3.x.x] 버튼을 눌러 다운로드 후 실행합니다.
③ [매우 중요] 설치 창 맨 아래에 있는 'Add python.exe to PATH'를 
   반드시 마우스로 클릭해서 체크(V)해 주세요.
④ 'Install Now'를 눌러 설치를 완료합니다.

--------------------------------------------------
2. 필수 부품(라이브러리) 설치 (최초 1회만 진행)
--------------------------------------------------
① 키보드의 [Windows 키 + R]을 동시에 누릅니다.
② 실행 창이 뜨면 cmd 라고 적고 엔터를 누릅니다.
③ 까만 창이 나오면 아래 글자를 마우스로 복사해서 붙여넣고 엔터를 누릅니다.
   
   pip install pywin32 openpyxl

④ 영어로 Successfully installed... 라는 문구가 뜨면 창을 닫습니다.

--------------------------------------------------
3. 파일 준비하기
--------------------------------------------------
바탕화면 등에 새 폴더를 하나 만들고, 아래 파일들을 한 곳에 모아주세요.

1) Auto_Award_Generator.py (실행 프로그램 본체)
2) Award_Data.xlsx (대상자 명단 엑셀 파일)
3) Award_Template.hwp (표창장 한글 템플릿 원본)
4) 표창장_자동생성기_실행.bat (마우스 더블클릭 실행용 버튼 활용)

--------------------------------------------------
4. 데이터 및 템플릿 작성 규칙
--------------------------------------------------
[엑셀 파일 규칙]
첫 번째 줄의 이름은 반드시 아래와 똑같아야 합니다.
(문서번호 / 표창명 / 부서 / 이름 / 내용 / 날짜 / 대표이사명 )

* [작업 종료 조건]: 엑셀을 읽어 내려가다 '문서번호'가 비어있는 칸을 
  발견하면, 데이터가 끝난 것으로 간주하고 즉시 작업을 종료합니다.
  (중간에 빈 줄이 끼어있지 않도록 주의해 주세요!)
* [안내문 유지]: '내용', '날짜', '대표이사명' 등의 칸을 비워두면(Null), 
  한글 템플릿에 원래 적혀있던 안내 문구가 지워지지 않고 예쁘게 유지됩니다.
* [중복 방지]: 이미 만들어진 사람은 '결과' 칸에 '성공' 또는 
  '성공(일부누락)'으로 자동 기록되며, 다음 실행 시 안전하게 건너뜁니다.

[한글 파일 규칙]
글자가 들어갈 자리에 '누름틀(Ctrl+K, E)'을 만들고, 필드 이름을 
반드시 아래의 영어(소문자)로 적어주세요.
- 문서번호: doc_no
- 표창명: award_name
- 부서: department
- 이름: name
- 내용: content
- 날짜: award_date
- 대표이사명: ceo

--------------------------------------------------
5. 프로그램 실행하기
--------------------------------------------------
[방법 A: 간편 실행]
미리 만들어둔 '표창장_자동생성기_실행.bat' 이 있다면 마우스로 더블클릭합니다.

[방법 B: 수동 실행]
① 폴더 주소창을 클릭하고 cmd 를 입력해 까만 창을 띄웁니다.
② 아래 글자를 적고 엔터를 누릅니다.
   > python Auto_Award_Generator.py

* [주의]: 엑셀 파일이 열려있으면 저장이 불가능해 에러가 납니다. 
  반드시 엑셀 창을 닫고 실행해 주세요.
* 작업이 끝나면 '결과물' 폴더 안에 완성된 표창장들이 들어있습니다.

==================================================
* 💡 [관리자 팁] 항목(컬럼)을 새로 추가하고 싶다면?
파이썬 코드 파일을 메모장으로 열고, 맨 위쪽 CONFIG 영역의 
"FIELD_MAPPING" 부분에 [엑셀 제목: 한글 필드명]을 한 줄만 
추가해 주면, 코드를 고치지 않아도 프로그램이 알아서 작동합니다!

* 🚨 에러가 났을 때 확인사항
- 프로그램 중간 멈춤 : 엑셀 중간에 '문서번호'가 없는 빈 줄이 있습니다.
- "1페이지 초과" 에러 : 내용이 너무 길어 2장으로 넘어간 경우입니다. 
- 값이 빈칸으로 나옴 : 한글 템플릿의 누름틀 이름(영어) 오타를 확인하세요.
