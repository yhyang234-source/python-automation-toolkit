==================================================
        표창장 자동 생성 시스템 사용 설명서
==================================================

이 프로그램은 엑셀(Excel)에 정리된 대상자 명단을 읽어와, 
한글(HWP) 표창장 템플릿에 자동으로 데이터를 입력하고 
개별 파일로 생성해 주는 자동화 프로그램입니다.

아래 순서대로 천천히 따라해 주세요.

--------------------------------------------------
1. 파이썬(Python) 설치하기 (최초 1회만 진행)
--------------------------------------------------
이 프로그램이 돌아가기 위한 '엔진'을 설치합니다.

① 인터넷 창을 열고 아래 주소로 접속합니다.
   https://www.python.org/downloads/
② 노란색 [Download Python 3.x.x] 버튼을 눌러 다운로드 후 실행합니다.
③ [매우 중요] 설치 창 맨 아래에 있는 'Add python.exe to PATH'를 
   반드시 마우스로 클릭해서 체크(V)해 주세요.
④ 'Install Now'를 눌러 설치를 완료합니다.

--------------------------------------------------
2. 필수 부품(라이브러리) 설치 (최초 1회만 진행)
--------------------------------------------------
① 키보드의 [Windows 키 + R]을 동시에 누릅니다.
② 실행 창이 뜨면 cmd 라고 적고 엔터를 누릅니다.
③ 까만 창이 나오면 아래 글자를 마우스로 복사해서 붙여넣고 엔터를 누릅니다.
   
   pip install pywin32 openpyxl

④ 영어로 Successfully installed... 라는 문구가 뜨면 창을 닫습니다.

--------------------------------------------------
3. 파일 준비하기
--------------------------------------------------
바탕화면 등에 새 폴더를 하나 만들고, 아래 3개 파일을 한 곳에 모아주세요.

1) Auto_Award_Generator.py (실행 프로그램)
2) Award_Data.xlsx (대상자 명단 엑셀 파일)
3) Award_Template.hwp (표창장 한글 템플릿 원본)

--------------------------------------------------
4. 데이터 및 템플릿 작성 규칙
--------------------------------------------------
[엑셀 파일 규칙]
첫 번째 줄의 이름은 반드시 아래와 똑같아야 합니다.
(문서번호 / 표창명 / 부서 / 이름 / 내용 / 날짜 / 대표이사명 / 결과)

* 중요: '문서번호'는 대상을 식별하는 필수 키값입니다. 반드시 입력해 주세요.
* 중요: '내용', '날짜', '대표이사명' 칸이 비어(Null) 있으면, 
  한글 템플릿에 설정된 원래 안내 문구가 그대로 유지됩니다.
* 이미 표창장이 만들어진 사람은 '결과' 칸에 '성공'이라고 
  자동으로 적히며, 다음 실행 시 안전하게 건너뜁니다.

[한글 파일 규칙]
글자가 들어갈 자리에 '누름틀'을 만들고, 필드 이름을 
반드시 아래의 영어(소문자)로 적어주세요.
- 문서번호: doc_no
- 표창명: award_name
- 부서: department
- 이름: name
- 내용: content
- 날짜: award_date
- 대표이사명: ceo

--------------------------------------------------
5. 프로그램 실행하기
--------------------------------------------------
① 폴더 주소창을 클릭하고 cmd 를 입력해 까만 창을 띄웁니다.
② 아래 글자를 적고 엔터를 누릅니다.
   
   python Auto_Award_Generator.py

③ [주의] 엑셀 파일이 열려있으면 에러가 납니다. 
   반드시 엑셀 창을 닫고 실행해 주세요.
④ 작업이 끝나면 폴더 안에 '결과물' 폴더가 생기고, 
   그 안에 완성된 표창장들이 들어있습니다.

==================================================
* 에러가 났을 때 확인사항
- "문서번호 누락" 에러 : 엑셀에 문서번호가 비어있습니다. 번호를 채워주세요.
- "1페이지 초과" 에러 : 내용이 너무 길어 2장으로 넘어간 경우입니다. 
  내용을 줄이거나 한글 템플릿 여백을 수정해 주세요.
- 값이 빈칸으로 나옴 : 한글 템플릿의 누름틀 이름(영어)에 
  오타가 없는지 다시 확인해 주세요.
- 파일명 오류 : 부서명 등에 슬래시(/) 등 특수문자가 있어도 
  프로그램이 자동으로 안전하게 변환하여 저장하니 안심하세요.
==================================================
